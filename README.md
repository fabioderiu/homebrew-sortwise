# Homebrew Tap for Sortwise

AI-powered file organizer for macOS.

## Install the CLI

```bash
brew tap fabioderiu/sortwise
brew install sortwise
```

## Install the Menu Bar App

```bash
brew tap fabioderiu/sortwise
brew install --cask sortwise
```

## More Info

See [fabioderiu/sortwise](https://github.com/fabioderiu/sortwise) for documentation.
